## 📖 Overview

This repository contains an exploratory data analysis (EDA) of the Netflix library, exploring the distribution of movies and TV shows across different countries, genres, and release years. The goal of this analysis is to uncover trends in content strategy, regional focus, and the evolution of the streaming giant's catalog over time.

## ⚙️ Tech Stack & Dependencies

- **Language**: Python 3.7+
- **Data Manipulation**: Pandas, NumPy
- **Data Visualization**: Matplotlib, Seaborn

## 🚀 Setup & Installation

To explore the analysis on your own machine:

### 1. Clone the Repository
```bash
git clone https://github.com/riteshv7/Netflix-Movies-and-TV-shows-Analysis.git
cd Netflix-Movies-and-TV-shows-Analysis
```

### 2. Install Dependencies
```bash
pip install pandas numpy matplotlib seaborn jupyter
```

### 3. Run the Notebook
The `netflix_titles.csv` dataset is already included in this repository. Just launch Jupyter:
```bash
jupyter notebook
```
Open `Netflix_Data_Analysis.ipynb` and run the cells!

## 📊 Analytics Highlights
- Comparisons of Movie vs. TV Show volume over the last two decades.
- Geospatial and regional analysis of where Netflix content is produced.
- Textual analysis of descriptions and genres to identify popular themes.
